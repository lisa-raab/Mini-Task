# Mini Task
## PCB Design
* **Software:** KiCad
* **Layers:** 2-layer PCB
* **Components:** four TMP118, one Molex PicoBlade 53048-0410, four capacities (10nF) and two resistors (5k&Omega;)
* **Design:** 20mm x 20mm with mounting holes for secure chassis integration and enclosure fitment
  
## Enclosure Design (CAD)
* **Software:** Autodesk Fusion
* **Features:** Accessible molex connector, lid is secured using screws, mounting holes, ventilation slots, overall slim design
